# {TITLE: One concept, one note}

## Usage

1. Copy this template into the `/notes` directory  
2. Rename using the convention:  
   `ZN-{YYYYMMDD}T{HHmmss}-{AUTHOR}-1.00.md`  
3. Ensure the note contains ONLY ONE atomic idea  
4. Write the title as a single concept (noun phrase, not a sentence)  
5. Fill the YAML metadata fields completely  
6. Write the Core Idea in your own words (max ~1 paragraph)  
7. Add an Insight / Claim only if the idea produces a conclusion or hypothesis  
8. Record the source of the idea (experiment, paper, log, observation)  
9. Link related concepts using [[wikilinks]]  
10. Keep the note self-contained but connected (no dependency on external explanation)  
11. Commit to Git when the idea is stable enough to be reused or referenced

## Core Idea  
Write the main idea in your own words.

## Key Points

## Key Insight
